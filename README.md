## Usage
How to use my plugin.

## Installation
Add this line to your application's Gemfile:

```ruby
gem 'wdw_frontend'
```

And then execute:
```bash
$ bundle
```

Or install it yourself as:
```bash
$ gem install wdw_frontend
```
